#  Mental Health Risk Prediction Web App

An AI-driven mental health assessment system that predicts **stress, depression, and emotional state** using **text, images, and behavioral analysis**.  
This project helps users understand their mental well-being and provides actionable insights through **multi-modal machine learning models**.
